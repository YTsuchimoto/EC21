新しい演算子として、3つの個体からなる交叉を考えました。
今回の実装では一様交叉になっていて、各個体の遺伝子座ごとに、以下の3つのパターンをランダムに選ぶという仕様になっています

パターン1:変化なし
パターン2:子1 ← 親2
　        子2 ← 親3
　        子3 ← 親1
パターン3:子1 ← 親3
　        子2 ← 親1
　        子3 ← 親2

triCrossOverTest.pyを実行すると、ランダムな値で初期化された3つの親個体と、それらを交叉させた3つの子個体が出力されます

今回は非常に単純な範囲でしか実装していませんが、一点交叉でパターン2や3を適用したり、子に対して3つの親のいずれかの遺伝子を確率的に遺伝させるなども考えられます。
またより一般的には交叉する個体の数をパラメータや遺伝子にするなどの発展も考えられます。
